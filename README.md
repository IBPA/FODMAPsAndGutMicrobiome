# Dietary FODMAPs, Gut Microbiome and Health in Irritable Bowel Syndrome
**Goal:** To investigate the interplay between dietary FODMAPs, gut microbiome and health in irritable bowel syndrome (IBS) using data from multiple studies.

## Code Architecture
```
├─ installation/

   ├─ ...
├─ integrated_analysis/

   ├─ Fig3.R
   ├─ Fig4.R
   ├─ Fig5.R
├─ run_all.sh
```
